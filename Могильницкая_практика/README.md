# Подсчет багажа на ленте в аэропорту

Система автоматического детектирования и подсчета багажа с использованием искусственных нейронных сетей.

## Описание

Веб-приложение для подсчета багажа на ленте в аэропорту с использованием предобученной модели YOLOv8. Система позволяет:
- Загружать изображения для обработки
- Использовать камеру для захвата изображений в реальном времени
- Детектировать багаж (чемоданы, сумки, рюкзаки) на изображениях
- Сохранять историю запросов
- Генерировать отчеты в форматах PDF и Excel

## Технологии

- **Backend**: Flask (Python)
- **Frontend**: HTML, CSS, JavaScript
- **Модель ИИ**: YOLOv8 (Ultralytics)
- **Обработка изображений**: OpenCV
- **Генерация отчетов**: ReportLab (PDF), Pandas/OpenPyXL (Excel)

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. При первом запуске модель YOLOv8 будет автоматически загружена.

3. Запустите приложение:
```bash
python app.py
```

4. Откройте браузер и перейдите по адресу: `http://localhost:5001`

## Использование

1. **Загрузка изображения**: Нажмите "Выбрать файл" и выберите изображение с багажом
2. **Использование камеры**: Нажмите "Включить камеру", сделайте снимок, затем обработайте
3. **Обработка**: Нажмите "Обработать" для детектирования багажа
4. **Просмотр истории**: Нажмите "Загрузить историю" для просмотра всех запросов
5. **Генерация отчетов**: Используйте кнопки для создания PDF или Excel отчетов

## Архитектура нейронной сети

Используется предобученная модель YOLOv8n (nano версия) от Ultralytics, обученная на датасете COCO. Модель детектирует следующие классы объектов, связанных с багажом:
- Чемоданы (suitcase)
- Сумки (handbag, bag)
- Рюкзаки (backpack)

## Особенности

- Автоматическое детектирование багажа на изображениях
- Визуализация результатов с bounding boxes
- Сохранение истории всех запросов
- Генерация отчетов в PDF и Excel форматах
- Поддержка работы с камерой в реальном времени
- Современный и удобный веб-интерфейс
